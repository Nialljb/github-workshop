Collaborative Neuroimaging Workshop
===================================

Welcome to **Collaborative Neuroimaging: Mastering GitHub & VS Code for HPC Workflows**.

This workshop teaches neuroimaging researchers how to use version control, 
collaborative development, and modern development tools on HPC systems.

.. note::
   **Duration:** 3 hours  
   **Level:** Intermediate  
   **Prerequisites:** Basic Python, SSH access to HPC cluster
